# Hikki SDK

Hikki SDK — это мощный и гибкий набор инструментов для реверс-инжиниринга Android-приложений. SDK позволяет легко
внедрять механизмы перехвата вызовов (hooking), настраивать проксирования и обходить системы защиты (anti-tamper),
обеспечивая полный контроль над логикой приложения внутри экосистемы Hikki.

## Основные возможности

- **Core SDK**: Основной модуль логики мода, хуков и логики проксирования.
- **Ghost Framework**: Модуль для обхода защитных механизмов, включая подмену подписи и источника установки (installer
  spoofing).
- **Удаленное управление**: Гибкая настройка параметров приложения в реальном времени через интерфейс Hikki.
- **Инструментарий для хукинга**: Готовые решения для перехвата методов и модификаций поведения приложения.
