# Hikki SDK

Hikki SDK — это комплексный набор инструментов, разработанный для реверс-инжиниринга Android-приложений. Он
предоставляет мощные возможности для хукинга, проксирования и обхода механизмов защиты от несанкционированного доступа (
anti-tamper) для управления ими внутри приложения Hikki store.

## Особенности

- **Core SDK**: Централизованное управление настройками, хуками и логикой прокси.
- **Ghost Framework**: Продвинутые обходы защиты, включая подмену подписи и установщика.
- **Удаленные настройки**: Поддержка удаленной конфигурации через Hikki.
- **Хуки функций**: Упрощенный интерфейс для проверки функций приложения, статуса подписки и многого другого.

## Документация

Ознакомьтесь со следующими руководствами, чтобы начать работу:

- [**Руководство по интеграции**](docs/INTEGRATION_RU.md): Подробные инструкции по интеграции SDK в целевые приложения с
  использованием Smali, Frida или ручного патчинга.
- [**Руководство по компиляции**](docs/COMPILING_RU.md): Пошаговое руководство по сборке SDK и извлечению необходимых
  DEX-файлов.
- [**Руководство по настройке параметров**](docs/SETTINGS_GUIDE_RU.md): Информация о том, как определять, управлять и
  локализовать настройки SDK.

## Быстрый старт

1. **Соберите SDK**: Следуйте [Руководству по компиляции](docs/COMPILING_RU.md), чтобы создать DEX-файлы.
2. **Настройте**: Настройте поведение SDK, используя [Руководство по настройке параметров](docs/SETTINGS_GUIDE_RU.md).
3. **Интегрируйте**: Используйте [Руководство по интеграции](docs/INTEGRATION_RU.md) для внедрения SDK в целевой APK.
