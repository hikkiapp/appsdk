# Руководство по компиляции

Это руководство объясняет, как скомпилировать `HikkiSdk` и извлечь полученные DEX-файлы для интеграции.

## Сборка и извлечение

Самый эффективный способ сборки SDK — использование задач извлечения. Эти задачи автоматически запускают компиляцию (
assemble), а затем извлекают файлы `.dex` из сгенерированного APK.

### 1. Релизная сборка (Release)

Предназначена для использования в продакшене. Включает полную оптимизацию R8 и удаляет все логи.

```bash
./gradlew extractReleaseDex
```

`./gradlew extractReleaseDex`

**Выходная директория:** `app/build/outputs/apk/release/dex/`

### 2. Бета-сборка (Beta)

Предназначена для тестирования. Сохраняет логи и использует упрощенную конфигурацию Proguard для облегчения отладки.

```bash
./gradlew extractBetaDex
```

`./gradlew extractBetaDex`

**Выходная директория:** `app/build/outputs/apk/beta/dex/`

---

Если вам нужен только APK-файл без извлечения DEX-файлов, вы можете запустить:

- `./gradlew assembleRelease`
- `./gradlew assembleBeta`
